# Frontend Service

This is the React frontend for the Agentic Blog Generation SaaS, built using Vite, TypeScript, Tailwind CSS, and Redux Toolkit.

## Prerequisites
- Node.js 18+

## Development Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Set environment variables (e.g., `VITE_API_BASE_URL` if not using proxy).
3. Start the development server:
   ```bash
   npm run dev
   ```

## Build
```bash
npm run build
```
